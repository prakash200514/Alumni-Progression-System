Project Overview

The Alumni Progression System is a web-based application designed to track and manage the career progress of students after graduation. It allows former students to log in and update their current status such as higher studies, employment, or other activities.

This system helps institutions maintain updated alumni records and analyze student progression over time.

🚀 Features
🔐 Student Login & Authentication
📝 Profile Update (Job / Higher Studies / Not Working)
📊 Track Student Career Progress
📅 Year-wise Status Updates
🗂️ Secure Data Storage
📈 Easy Monitoring for Institutions

🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL

🏗️ System Architecture
User accesses the system via browser
Student logs in using credentials
Dashboard displays current profile
Student updates career status
Data is stored in database
Admin (optional) can view reports
